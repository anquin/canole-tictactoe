Canole Tic Tac Toe
===================

A Tic-Tac-Toe game that never loses, made with C and OpenGL.

"Canole" stands for "C ANd OpengL" and also "CAnNOt LosE"

![tic-tac-toe](ttt.png)

Build
-----

*On GNU/Linux:*

Requires freeglut to be installed, then:

    $ ./builddeps.sh
    $ make
